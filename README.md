1️⃣ What is the difference between var, let, and const?
    var is function-scoped and it can be redeclared,
    let is block-scoped and can be reassigned but not redeclared,
    const is block-scoped, it cannot be reassigned.


2️⃣ What is the spread operator (...)?
    It expands elements of an array or object into individual values.

3️⃣ What is the difference between map(), filter(), and forEach()?
   map() create a new array with transforming each element,
   filter() creates a new array with elements which pass a condition,
   forEach() simply loops by elements without returning a new array.

4️⃣ What is an arrow function?
   It is a shorter syntax for writing functions in JavaScript using =>

5️⃣ What are template literals?
   Template literals are strings written with backticks ` ` that allow embedded variables and expression using ${}.
